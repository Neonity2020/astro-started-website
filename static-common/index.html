简介: Astro 群岛

Astro群岛代表了前端web架构的领先范式转变。Astro提取你的UI变成更小的，独立的组件。将未使用的JavaScript替换为轻量级HTML，保证更快的加载和交互时间(TTI)。

详细了解

  • 零JavaScript运行时间

    Astro在服务器上呈现HTML，并去除任何剩余的、未使用的JavaScript。

  • 岛屿的力量

    需要交互式UI?单独加载，非阻塞 平行组成岛屿。

  • 延迟加载的岛屿

    组件只有在滚动进入视图时才会混合。如果用户从未看到它，它永远不会被加载。

为内容而设计

你的内容，你选择。

Astro与您喜爱的内容源一起工作。从文件系统中提取或获取内容 从您喜爱的CMS、数据库或API远程访问。Astro支持静态输出(SSG)和 实时服务器输出(SSR)，可以按需呈现您的内容。

Using your CMS With top-level await, Astro makes it easy to fetch content from your CMS.
Creating pages with Markdown Creating a page using Markdown is easy with Astro! Add a .md file to your pages directory and Astro will turn it into a page on your site. You can specify an Astro component in frontmatter to use as a page layout.
Creating pages with MDX Creating a page using MDX is easy with Astro! Add a .mdx file to your pages directory and Astro will turn it into a page on your site. MDX files make it easy to mix components from any framework with your text content.
Using an API With native fetch built-in, Astro makes it easy to work with data from any API. Powered by https://owen-wilson-wow-api.onrender.com
